# 今日食籤 🎋｜減脂減醣菜單抽籤

不知道今天吃什麼？讓籤筒決定。

依照營養師規劃的三份減脂減醣外食菜單（1600／1800／2000 大卡）製作的抽籤工具。選好熱量、按下抽籤，就從配好的組合裡抽出今天的早餐、午餐、晚餐和點心。

**線上使用：** https://huawiz.github.io/meal-lottery/

## 功能

- **三種熱量**：1600／1800／2000 大卡，切換時顯示各自的六大類份數（澱粉、豆魚蛋肉、油脂、蔬菜、水果、乳品）
- **抽一整天或單一餐別**：一整天可選兩種抽法
  - **照組合抽**（預設）：從營養師配好的一日組合中整欄抽出（每個熱量 15 組、共 45 組），份數配置與原始菜單一致，並標示來源組合
  - **自由混搭**：各餐獨立抽，變化更多
- **指定商家**：今天附近只有某幾家？勾選商家後只從這些店的選項抽，並即時顯示各餐別剩餘可抽數量。通用點心（水果、無糖豆漿等）不受篩選影響
- **重抽單一籤**：整天抽出後，不想吃的那餐可以單獨重抽
- **輸出今日籤**：把當日菜單產生成一張含日期的圖片，手機可直接分享或存到相簿，方便記錄或傳到群組
- **表格模式**：依原始菜單排版瀏覽全部內容（3 組 × 組合一～五），方便核對

## 部署

純前端、單一檔案，React 已內嵌，無任何外部相依（字型除外，離線時退回系統字體）。

- **GitHub Pages**：把 `index.html` 放在 repo 根目錄，Settings → Pages 選擇 `main` / `(root)` 即可
- **本機使用**：直接雙擊 `index.html` 用瀏覽器開啟
- **手機**：瀏覽器開啟網址後「加入主畫面」，即可像 App 一樣使用

## 開發

- `meal-lottery.jsx`：原始碼（React 元件，含全部菜單資料）
- `index.html`：部署用檔案，由 jsx 經 Babel 編譯後與 React 18 UMD 一起打包而成

菜單資料在 jsx 的 `RAW` 常數中，結構對應原始菜單（每個熱量 3 組菜單表 × 組合一～五）。修改菜單後需重新編譯：

```bash
npm install @babel/cli @babel/core @babel/preset-react
npx babel meal-lottery.jsx --presets @babel/preset-react -o app.compiled.js
# 再將編譯結果與 React UMD 一起嵌入 index.html
```
